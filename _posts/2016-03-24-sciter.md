# class 引起的惨案 #

Legacy javac toolchain:

	javac (.java --> .class) --> dx (.class --> .dex)

New Jack toolchain:

	Jack (.java --> .jack --> .dex)

由此可见，jack 不会产生 class文件，所以基于 class的lint等工具将无法工作。

包括依赖class的工具以及自己的编译脚本等。

# Proguard 可能不再需要 #

Jack 工具包含了compile，shrinking，obfuscation，repackaging，multidex 。所以呢独立的proguard将不在需要。
